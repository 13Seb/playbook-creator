PBCPlay::PBCPlay ( const PBCPlay &  other)

The copy constructor.

The name, code name and categories of the play are simply copied. The formation needs special treatment because the copy constructor of PBCFormation does not copy the players routes and motions but only their positions. So we need to set the routes and motions here manually.
