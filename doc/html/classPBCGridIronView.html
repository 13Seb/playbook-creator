Detailed Description

A canvas class that paints a grid iron on it.

This class extends a QGraphicsscene. An instance of this class can be seen as a canvas on which it paints a grid iron itself. In this context a grid iron is an american football field, consisting of a few lines (line of scrimmage, 5yd marking line, 10yd marking line,...) and a ball (the football)
