Detailed Description

This is a data model class representing a playbook.

PBCPlaybook is a Singleton class (it inherits PBCSingleton). Thus the application handles only one playbook at a time. It stores the plays, formations, routes, and categories.

bool PBCPlaybook::addRoute ( PBCRouteSP  route,
bool  overwrite = false 
)

Adds a route to the playbook.

Actually this function adds a pointer to a route. When overwriting a route later, this pointer is used so that the routes which have the same name will be overwritten in all plays. So you only have to change a route one time and it is changed automatically in the whole playbook

Parameters
routeThe route to add
overwriteSpecifies if a route with the same name should be overwritten.

If overwrite is false and the route has the same name as an formation that has been added to the playbook already, then the new route is not added to the playbook. If overwrite is true, a route with the same name is overwritten by the new route.

Returns
true if the route has been successfully added to the playbook
